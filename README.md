# How to run the application

## Backend
- ``` cd backend ```
- Have postgres up and running.
- Set database credentials in .env file
- Run ``` npm install ```
- Run ``` npm run dev ```
- Server is started at ``` http://localhost:5000 ```

## Frontend
- ``` cd frontend ```
- Run ``` npm install ```
- Run ``` npm start ```
- Client is started at ``` http://localhost:3000 ```
